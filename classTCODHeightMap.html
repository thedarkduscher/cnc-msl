Class TCODHeightMap

heightmap Roguelike toolkits Heightmap toolkit This toolkit allows to create a 2D grid of float values using various algorithms.
The code using the heightmap toolkit can be automatically generated with the heightmap tool (hmtool) included in the libtcod package.
