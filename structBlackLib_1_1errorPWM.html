Struct BlackLib::errorPWM

This struct holds PWM errors and includes pointer of errorCorePWM struct.

Public Members

errorCorePWM * pwmCoreErrors
This struct initializes at constructor in BlackPWM class. Its value can set with function in BlackCorePWM class.
bool periodFileError
Its value can change, when opening pwm's period value file, at functions in BlackPWM class.
bool dutyFileError
Its value can change, when opening pwm's duty value file, at functions in BlackPWM class.
bool runFileError
Its value can change, when opening pwm's run value file, at functions in BlackPWM class.
bool polarityFileError
Its value can change, when opening pwm's polarity value file, at functions in BlackPWM class.
bool outOfRange
Its value can change, when setting some variables of pwm, at functions in BlackPWM class.

Public Functions

errorPWM ()
This function clears all flags and initializes errorCorePWM struct.
errorPWM (errorCorePWM *base)
This function clears all flags and assigns input parameter to pwmCoreErrors variable.
