Struct BlackLib::errorGPIO

This struct holds GPIO errors and includes pointer of errorCoreGPIO struct.

Public Members

errorCoreGPIO * gpioCoreErrors
This struct initializes at constructor in BlackGPIO class. Its value can set with function in BlackCoreGPIO class.
bool exportError
Its value can change, when checking GPIO pin's export status, at functions in BlackGPIO class, if working mode is secureMode.
bool directionError
Its value can change, when checking GPIO pin's direction, at functions in BlackGPIO class, if working mode is secureMode.
bool readError
Its value can change, when opening gpio pin's value file, at functions in BlackGPIO class.
bool writeError
Its value can change, when opening gpio pin's value file, at function in BlackGPIO class.
bool forcingError
Its value can change, when trying to write something to input type gpio pin, at function in BlackGPIO class.

Public Functions

errorGPIO ()
This function clears all flags and initializes errorCoreGPIO struct.
errorGPIO (errorCoreGPIO *base)
This function clears all flags and assigns input parameter to gpioCoreErrors variable.
