Class TCODZip

zip Base toolkits Compression toolkit This toolkit provides functions to save or read compressed data from a file. While the module is named Zip, it has nothing to do with the .zip format as it uses zlib compression (.gz format). Note that this modules has no python wrapper. Use python built-in zip module instead.
You can use the compression buffer in two modes: put data in the buffer, then save it to a file, load a file into the buffer, then get data from it.
